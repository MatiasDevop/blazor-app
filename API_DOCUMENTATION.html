📋 Phase 1-2: User Management & Authentication

POST /api/userprofile/student PUBLIC

Register a new student with work/education history

POST /api/userprofile/company PUBLIC

Register company and recruiter profile

POST /api/userprofile/careercenter PUBLIC

Register career center and administrator

GET /api/userprofile/{id} AUTH

Retrieve user profile by ID

GET /api/userprofile/email/{email} AUTH

Retrieve user by email (Auth0 integration)

PUT /api/userprofile/{id} AUTH

Update user profile information

💼 Phase 2: Job Applications

POST /api/jobapplications AUTH

Submit a job application

GET /api/jobapplications/user/{userId} AUTH

View user's job applications

📝 Phase 3: Job Posts CRUD

POST /api/jobposts RECRUITER

Create new job posting

GET /api/jobposts/{id} PUBLIC

Get job post details

POST /api/jobposts/search PUBLIC

Search jobs with filters, sorting, pagination

PUT /api/jobposts/{id} RECRUITER

Update job post (with nested collections)

DELETE /api/jobposts/{id} RECRUITER

Soft delete job post

GET /api/jobposts/recommended/{userId} AUTH

Get personalized job recommendations

🤝 Phase 4: Connection System

POST /api/connections AUTH

Send connection request

PUT /api/connections/{id}/accept AUTH

Accept pending connection

DELETE /api/connections/{id} AUTH

Decline/remove connection

GET /api/connections/user/{userId} AUTH

Get user's connections

💰 Phase 6: Discount System

POST /api/discounts ADMIN

Create promotional discount code

📄 Phase 7: Invoice Management

POST /api/invoices AUTH

Generate invoice with discount support
